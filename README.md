# PortfolioApp

PortfolioApp is a web application that showcases your portfolio. It allows users to display their projects, skills, and other relevant information.

## Tech I use
- C#
- .Net Framework(4.7.2)
- MVC
- Entity Framework
- MSSQL
- HTML/CSS/Bootstrap/JS
- DbFirst
